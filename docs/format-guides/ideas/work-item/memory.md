Tag with `work-item` + area. Check brain://overview for existing tags first.

Write as a focused task with concrete steps:

> Why: what plan/phase this belongs to
> What: specific deliverable
> Who: who does it
> How: approach, blockers

- [ ] Step 1
- [ ] Step 2
- [ ] Verify / test
- [ ] Done — update parent plan

Keep it small. If more than 5-6 checkboxes, split into multiple work-items.
